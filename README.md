Vector utilities for 2, 3, and 4 element vectors, all inline - by Jean-Francois Doue, Paul S. Heckbert, and J. Nagle (1993).


## Installation

Run:
```bash
$ npm i algebra3.cxx
```

And then include `algebra3.h` as follows:
```c
#include "node_modules/algebra3.cxx/algebra3.h"
```
